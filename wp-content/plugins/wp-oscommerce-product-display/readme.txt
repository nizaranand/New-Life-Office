== Description ==

WP OSCommerce Product Display is a WordPress plugin that allows you to show products
from your separate OSCommerce store in your WordPress site.

It is based on a tutorial on NetTuts by C. Lupu. However it has been extended and 
turned into a widget which you can add into your sidebar and change the amount of
products you wish to show.


== Installation ==

1. Upload the the plugin folder to the '/wp-content/plugins/' directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Enter your store database details in the 'Settings > Product Generator' window.
1. Add the 'Product generator' widget to one of your sidebars.

== Frequently Asked Questions ==

= Will this work for any online store or only OSCommerce stores? =

No, this plugin was built to only work with OSCommerce stores.

= What options can I change from the widget settings? =

You can give the widget a title and select the number of products to generate. More options to be added shortly.
